# local-llama2


### Step 1: Create a virtual environment
```bash
conda create -p venv python==3.10 -y
```

### Step 2: Activate the virtual environment
```bash
conda activate venv/
```

### Step 3: Install the dependencies

```bash
pip install -r requirements.txt
